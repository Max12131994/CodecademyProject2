CSS Table Reference

- CSS Table Tags -

Name Description Example
color: ; The color CSS data type represents a color. color: black;
text-align: ; The text-align property specifies the horizontal alignment of text in an element. text-align: center;
text-align: right;
text-align: left;
font-family: ; The font-family property specifies the font for an element. The font-family property can hold several font names as a "fallback" system. If the browser does not support the first font, it tries the next font. font-family: "Chewy", monospace, serif;
font-size: ; The font-size property sets the size of a font. font-size: 20px;
font-size: 100%;
